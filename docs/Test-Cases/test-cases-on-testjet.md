# Test Cases on TestJet

In this section, you will see a list of the total test cases, tags, and run results of a particular project. You will be able to search for a particular test case and also filter test cases by time duration from the **“Filter by”** drop-down. On the right panel you will see buttons **“Create group”**, **“Clone”**, **“Create Test Run”**, **“Delete”** and **“New Test Case”**.
