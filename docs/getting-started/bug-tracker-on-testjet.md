# Bug Tracker on TestJet

You can integrate a JIRA account with TestJet under any paid subscription to keep track of the failed test cases. 



![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/151010754453/original/FrRh4Ze-cPeplW7I1L5LDOYfWP7v3iaaEg.png?1675754069)

  


Steps:

- The user must be signed in to their JIRA account.

- Click on the “JIRA” icon 

- Enter a valid host name 

- Click on “login”
